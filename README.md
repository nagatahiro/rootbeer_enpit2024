## エレベーターピッチ
**旅行先で割り勘**をしたい<br>
**大学生グループ**向けの<br>
**ワリポケ**👛というプロダクトは<br>
**旅行の最後に誰が誰にいくら支払うかをまとめて計算してくれるアプリ**です。<br>
これは**レシートを撮ることで割り勘をする**ことができ<br>
従来の**割り勘アプリ**とは違って、<br>
**グループで共有して自動で管理してくれる機能**が備わっている。
